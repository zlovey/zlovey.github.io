迭代器

序列，字典与集合等类型都可以看做是一个容器，用来存放多个元素，并且每种类型都提供了相应的方法来操作容器中的元素。这些类型都可以用在for循环中进行遍历，依次获取容器中的每一个元素。这些可以用在for循环中进行遍历的对象称为可迭代对象。

能被for 的就是可迭代对象，能next() 的是迭代器，list tuple 是可迭代对象，但不是迭代器。
