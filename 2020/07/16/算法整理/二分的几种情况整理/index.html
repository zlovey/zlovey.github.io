搜索一个数是否存在

细节部分

  • 左闭右闭区间
  • left<=right
  • 把target相等，小于和大于三种情况分开
  • mid+1和mid-1处理边界

left<=right与left<right

  • 原理理解：之所以写成left<=right，这样结束的条件是left>right，这样作为一个区间，其不可能存在有效元素了。而如果使用left<right此时的结束条件为left=right，此时中间还有一个数nums[left]，而这个数其实有可能是我们需要寻找的数，因此可能会漏掉部分结果。

  • 如果要使用left<right的形式，就是在跳出循环的时候left=right此时再加一个判断，即跳出循环后再对当前数加一个判断就好了。

left = mid + 1，right = mid - 1与 right = mid，left = mid

  • 由于是闭区间，且mid已经不是我们要找的元素，因此可以吧mid排除掉。

寻找左侧边界的二分搜索

细节部分

  • 左闭右开区间
  • left<right
  • 当mid与target相等时将mid值作为右边界。
  • mid+1和mid处理边界

left<right

  • 由于每次搜索的区间为左闭右开，因此到循环终止条件left==right时，此时的搜索区间为[left,left)，所以可以正确终止。

在什么时候return -1表示没有该元素

  • 对于这个数组，算法会返回 1。这个 1 的含义可以这样解读：nums 中小于 2 的元素有 1 个。
  • 比如对于有序数组 nums = [2,3,5,7], target = 1，算法会返回 0，含义是：nums 中小于 1 的元素有 0 个。
  • 再比如说 nums = [2,3,5,7], target = 8，算法会返回 4，含义是：nums 中小于 8 的元素有 4 个。
  • 综上可以看出，函数的返回值（即 left 变量的值）取值区间是闭区间 [0, nums.length]，所以我们简单添加两行代码就能在正确的时候 return -1：

left = mid + 1，right = mid

  • 因为搜索区间是左闭右开，当mid过了之后，被分割成的新的区间应该为[left,mid)和[mid+1,right)

为什么可以得到左边界

因为在mid等于目标值的时候并没有立即返回，而是不断收缩区间的上界

最后返回left和right都一样，因为相等

寻找右侧边界的二分查找

细节部分

  • left<right
  • 当mid与target相等时将mid+1值作为左边界。
  • mid+1和mid处理边界
  • 返回left-1

left<right

  • 同上
  • 由于每次搜索的区间为左闭右开，因此到循环终止条件left==right时，此时的搜索区间为[left,left)，所以可以正确终止。

在什么时候return -1表示没有该元素

  • 与之前的类似
  • while的终止条件为left==right，函数的返回值（即 left 变量的值）取值区间是闭区间 [0, nums.length]，所以我们简单添加两行代码就能在正确的时候 return -1：

left = mid + 1，right = mid

  • 同上
  • 因为搜索区间是左闭右开，当mid过了之后，被分割成的新的区间应该为[left,mid)和[mid+1,right)

为什么可以得到右边界

因为在mid等于目标值的时候并没有立即返回，而是不断收缩区间的下界

最后返回left-1

至于为什么要减一，这是搜索右侧边界的一个特殊点，关键在这个条件判断：

因为我们对 left 的更新必须是 left = mid + 1，就是说 while 循环结束时，nums[left] 一定不等于 target 了，而 nums[left-1] 可能是 target。

给你一个 山脉数组 mountainArr，请你返回能够使得 mountainArr.get(index) 等于 target 最小 的下标 index 值。

如果不存在这样的下标 index，就请返回 -1。

何为山脉数组？如果数组 A 是一个山脉数组的话，那它满足如下条件：

首先，A.length >= 3

其次，在 0 < i < A.length - 1 条件下，存在 i 使得：

  • A[0] < A[1] < ... A[i-1] < A[i]
  • A[i] > A[i+1] > ... > A[A.length - 1]

你将 不能直接访问该山脉数组，必须通过 MountainArray 接口来获取数据：

  • MountainArray.get(k) - 会返回数组中索引为k 的元素（下标从 0 开始）
