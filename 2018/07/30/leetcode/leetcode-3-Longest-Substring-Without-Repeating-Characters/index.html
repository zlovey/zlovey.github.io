leetcode.3 Longest Substring Without Repeating Characters

发表于 2018-07-30 | 更新于 2020-04-14 | 分类于 leetcode

问题描述：

Given a string, find the length of the longest substring without repeating characters.

Examples:

Given "abcabcbb", the answer is "abc", which the length is 3.

Given "bbbbb", the answer is "b", with the length of 1.

Given "pwwkew", the answer is "wke", with the length of 3. Note that the answer must be a substring, "pwke" is a subsequence and not a substring.

问题分析：题目要求给定字符串的最长未重复子串，由于需要比较字符是否重复出现，故要将每个字符出现的位置记录下来，然而经过分析，只需要记录最近一次该字符所在的位置即可，所以定义变量T[256]作为位置数组并将其初始化为0，256是因为ACSII码有256个字符（包括键盘上有的和没有的）。

当字符未出现或者出现在所取子串的左边的时候，不影响所取子串的长度，将其加入子串中；而当其重复出现时，则将子串的left标记移到所记录的该字符上一个出现的位置之后。结果仅记录最长的子串长度即可。
