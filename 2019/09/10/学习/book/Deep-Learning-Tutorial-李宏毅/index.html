New activation function
  • Deeper usually does not imply better.
  • Vanishing Gradient Problem梯度消失问题
    • 在神经网络中，当前面隐藏层的学习速率低于后面隐藏层的学习速率，即随着隐藏层数目的增加，分类准确率反而下降了。这种现象叫做消失的梯度问题。
    • 从求导结果可以看出，Sigmoid导数的取值范围在0~0.25之间，而我们初始化的网络权值$|w|$通常都小于1，因此，当层数增多时，小于0的值不断相乘，最后就导致梯度消失的情况出现。同理，梯度爆炸的问题也就很明显了，就是当权值$|w|$过大时，导致 $|\sigma’(z)w| > 1$，最后大于1的值不断相乘，就会产生梯度爆炸。
  • ReLU（ Rectified Linear Unit ）
    • 激活函数的导数为1，不会出现梯度消失问题
    • 优点：1)不饱和；2)计算效率高；3)收敛速度快
  • 原型Maxout:
    • 即取最大的值作为输出：
    • ReLU is a special cases of Maxout
Adaptive Learning Rate

  • Set the learning rate η carefully

    • If learning rate is too large—>Total loss may not decrease after each update
    • If learning rate is too small—>Training would be too slow

  • Popular & Simple Idea: Reduce the learning rate by some factor every few epochs.

    • 训练开始时，距离最小点很远，于是选取较大的学习率
    • 经过一些epoch之后可以适当减小学习率的值

  • Adagrad——错误方向的阻力加大（一双不好走路的鞋子——使其想要快速到达终点）

    • 解决不同参数应该使用不同的更新速率的问题

    • $w← w-\eta_w \partial L/\partial w$

    • $\eta_w$是根据学习率变换的变量
      $$
      \eta_w=\frac{\eta}{\sqrt{\sum_{i=0}^{t}{(g^i)^2}}}
      $$
      $\eta$是常数，$g^i$是在完成第i次更新后的$𝜕L/𝜕w$值

  • RMSprop、Adam——一双不好走路的鞋+下坡（adam一般更好）

  • 其他的一些改善学习率的方法：

    • Adadelta 、Nadam等

Weight Decay(regularization L2正则化)
  • prunes out the useless link between neurons. 类似剪枝，对没有用或者用处很小的神经元进行删减
  • 图示

  • 实现(C是cost function)
    • $$ C = C_0 + \frac{\lambda}{2n} \sum_w w^2$$
    • $$ \frac{\partial C}{\partial w} = \frac{\partial C_0}{\partial w} + \frac{\lambda}{n} w $$
    • $w=w-\eta\frac{\partial C}{\partial w}$
    • 让weight变小一点，带来的好处是可以是整个神经网络对输入中的噪音（或者一点点变化）不要那么敏感；weight太大，其对应的输入的一点点变化就会起到主导作用，进而显著改变输出。
    • 从公式来看，weight decay对于比较大的weight，decay的更多，比较小的weight，decay较小；这就相当于，weight越大，惩罚越大，即可以更有效的减少Cost函数。
    • 让神经网络倾向于形成更简单的，“斜率”更小的模型；比如一个非线性模型，我们可以用很复杂的高阶多项式来表示，也可以容忍一些噪音，通过简单的低阶多项式来表示，甚至直接使用线性函数来表示。
Dropout

  • Each time before updating the parameters

    • Each neuron has p% to dropout ，让某个神经元的激活值以一定的概率p停止工作，这样可以使模型泛化性更强，因为它不会太依赖某些局部的特征

      • The structure of the network is changed.

    • Using the new network for training

  • For testing：

    • 如果训练集的drop率为p%，则测试集的神经元权重较训练集应该乘上(1-p%）——因为测试集不做drop

  • Dropout之后的网络结构不同，故会训练出不同的模型，通过average可以取得更好的结果

  • 可以解决过拟合的问题：

    • 减少神经元之间复杂的共适应关系： 因为两个神经元不一定会同时出现在一个网络里，故有利于加强他们自身更鲁棒的学习。
    • 取平均的作用：相当于对很多个不同的神经网络取平均。而不同的网络产生不同的过拟合，一些互为“反向”的拟合相互抵消就可以达到整体上减少过拟合。

Convolutional Neural Network (CNN)
  • Why CNN for Image（some properties）
    • When processing image, the first layer of fully connected network would be very large
    • Some patterns are much smaller than the whole image
    • Subsampling the pixels（减小像素） will not change the object
  • CNN的结构
    • Convolution:
      • Some patterns are much smaller than the whole image（通过鸟的喙形判断鸟的品种）
      • The same patterns appear in different regions.（出现在图片不同位置的鸟喙）
    • Max Pooling:
      • Subsampling the pixels will not change the object （可以对图片进行缩放处理，减少像素点，减少网络结构的大量参数）
  • Convolution卷积层

      • Padding填充：对数据做的操作。一般有两种，一种是不进行操作，一种是补0使得卷积后的激活映射尺寸不变。

        • Zero Padding零填充：有时，在输入矩阵的边缘使用零值进行填充，这样我们就可以对输入图像矩阵的边缘进行滤波。零填充的一大好处是可以让我们控制特征图的大小。使用零填充的也叫做泛卷积，不使用零填充的叫做严格卷积

          • 全0填充的含义，不是在右（下）多加一行（列）0。具体加多少0要根据实际情况决定

          • 使用零填充之后再使用filter 进行卷积可以使得输入和输出的维度相同

      • 其实我们输入的图像一般为三维，即含有R、G、B三个通道。但其实经过一个卷积核之后，三维会变成一维。它在一整个屏幕滑动的时候，其实会把三个通道的值都累加起来，最终只是输出一个一维矩阵。

  • Pooling池化层(池化层一般放在卷积层后面。所以池化层的输入是卷积层的输出)
    • 池化层也有padding的选项。跟卷积层一样的，在外围补0，然后再池化。
    • 最大池化 max-pooling（更常用）
      • 为了提取最明显的特征
    • 平均池化 mean-pooling
      • 顾及每一个像素，所以选择将所有的像素值都相加然后再平均
      • 即每个filter取平均值
  • Flatten（扁平化）
    • Flatten层用来将输入“压平”，即把多维的输入一维化，常用在从卷积层到全连接层的过渡。Flatten不影响batch的大小。
  • BP算法（error BackPropagation）是训练神经网络最为经典的算法
    • 基于梯度下降
Recurrent Neural Network (RNN)循环神经网络

  • Slot Filling

    • 填槽(Slot filling)指的是为了让用户意图转化为用户明确的指令而补全信息的过程

  • 比如在一个订票系统上，我们的输入 “Arrive Taipei on November 2nd” 这样一个序列，我们设置几个槽位（Slot），希望算法能够将关键词’Taipei’放入目的地（Destination）槽位，将November和2nd放入到达时间（Time of Arrival）槽位，将Arrive和on放入其他（Other）槽位，实现对输入序列的一个归类，以便后续提取相应信息。

    • 用前馈神经网络（Feedforward Neural Network）来解决这个问题的话，我们首先要对输入序列向量化，将每一个输入的单词用向量表示，可以使用 One-of-N Encoding 或者是 Word hashing 等编码方法，输出预测槽位的概率分布。

  • 但是这样做的话，有个问题就出现了。如果现在又有一个输入是 “Leave Taipei on November 2nd”，这里Taipei是作为一个出发地（Place of Departure），所以我们应当是把Taipei放入Departure槽位而不是Destination 槽位，可是对于前馈网络来说，对于同一个输入，输出的概率分布应该也是一样的，不可能出现既是Destination的概率最高又是Departure的概率最高。

  • 所以我们就希望能够让神经网络拥有“记忆”的能力，能够根据之前的信息（在这个例子中是Arrive或Leave）从而得到不同的输出。将两段序列中的Taipei分别归入Destionation槽位和Departure槽位。

  • RNN

    • The output of hidden layer are stored in the memory.

    • Memory can be considered as another input.

    • The same network is used again and again.

    • 整体结构

  • Bidirectional RNN（双向rnn）

  • Long Short-term Memory (LSTM)

    • 结构：

    • 参数结构：

      • 由Memory Cell， Input Gate， Output Gate， Forget Gate 组成
      • 特殊的神经元结构，包含4个input（三个Gate的控制信号以及输入的数据），1个output
      • 激活函数通常选用sigmoid function， sigmoid的输出介于0到1之间，表征了Gate的打开程度
      • Short-term，表示保留对前一时间点输出的短期记忆，相比于最原始的RNN结构中的记忆单元（每次有新的输入时记忆体的状态就会被更新，因此是短期的记忆），LSTM的记忆体拥有相对较长的记忆时间（由Forget Gate决定），所以是Long Short-term

  • 学习/训练过程 to pick the best function

    • 损失函数
      • 每一个时间点的RNN的输出和标签值的交叉熵（cross-entropy）之和
    • 使用被称作Backpropagation through time（BPTT）的梯度下降法
    • 训练其实是比较困难的，因为Total Loss可能会出现剧烈的抖动
      • 损失函数的表面要么非常平坦，要么非常陡峭
      • 原因：
      • 与激活函数是Sigmoid或者ReLU无关
      • 由于RNN采用时间序列的结构，权重值在不同时间点被反复使用，这种累积性的变化可能对结果造成极大的影响，也可能会很长一段时间保持平稳。（$1^{1000}=1;1.01^{1000}->20000$）
    • 处理方法：
      • LSTM
        • 可以处理梯度消失的问题（不能处理梯度爆炸）——只有在遗忘门关掉的时候记忆单元的值才会对输出失去影响
        • 可能会引起梯度爆炸的问题
      • GRU(Gated Recurrent Unit)

  • 应用

    • 多对一 Input is a vector sequence, but output is only one vector
      • 情感分析
    • 多对多Both input and output are both sequences,
      • but the output is shorter
        • 语音识别
        • CTC
      • with different lengths.
        • 机器翻译
    • 一对多 Input an image, but output a sequence of words
      • 字幕生成
