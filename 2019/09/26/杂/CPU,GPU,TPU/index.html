CPU,GPU,TPU

发表于 2019-09-26

CPU

  • 中央处理器（Central Processing Unit）

  • 作为计算机系统的运算和控制核心，是信息处理、程序运行的最终执行单元。

CPU 是基于冯诺依曼架构的通用处理器，可以为数百万的不同应用加载任何软件。我们可以使用 CPU 处理文字、控制火箭引擎、执行银行交易或者使用神经网络分类图像。正因为此，CPU依靠读取指令来完成与指令对应的操作，即每完成一次计算之后就将数据存入内存，读取指令之后再取出数据进行下一次计算，在机器学习的大规模运算中显得较为笨拙且耗时。

GPU

  • 图形处理器（Graphics Processing Unit）

  • 是一种专门在个人电脑、工作站、游戏机和一些移动设备（如平板电脑、智能手机等）上做图像和图形相关运算工作的微处理器

GPU 使用一种简单的策略：在单个处理器中使用成千上万个 ALU（算术逻辑单元）。现代 GPU 通常在单个处理器中拥有 2500-5000 个 ALU，意味着可以同时执行数千次乘法和加法运算，这种GPU的架构在大量并行化的应用中效果很好。然而GPU毕竟也是一种通用的处理器，必须支持几百万种不同的应用和软件。

GPU在CPU的基础上并没有解决读取一次指令并完成一次相应操作的冯诺依曼瓶颈问题，而是仅仅增加了算数逻辑单元数量，使得在一次运算中可以完成大量的计算，且同时也因为复杂的线路而增加GPU的物理空间占用。

TPU

  • 张量处理单元（Tensor Processing Unit）

  • 由谷歌设计，是一款为机器学习而定制的芯片，经过了专门深度机器学习方面的训练，它有更高效能（每瓦计算能力）

TPU相较于CPU和GPU，它并不是一种通用的处理器，而是专用的处理神经网络计算的单元，由于其并不需要处理很多软件功能，该处理器的主要任务是矩阵处理，于是可以对冯诺依曼瓶颈进行大幅度简化。在设计结构中放置成千上万的乘法器和加法器并将它们直接连接起来，以构建那些运算符的物理矩阵。这被称作脉动阵列（Systolic Array）架构，使得TPU可以一次性进行多次计算而不用存储及访问内存，因此可以在神经网络运算上达到高计算吞吐量，同时能耗和物理空间都很小。
