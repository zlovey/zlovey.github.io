激励函数简介

发表于 2019-11-05 | 更新于 2020-04-06 | 分类于 学习 ， mofan

为什么需要使用激励函数：

  • ——解决线性函数解决不了的问题

  • 普通线性函数为 y=Wx

  • 添加了激励函数的函数为 y=AF(Wx),其中AF即为激励函数，通常用sigmoid/relu/tanh三种函数，就可以把原来的线性函数掰弯==

  • 在模型层数不多的时候，三种激励函数都可以随意选择；然而在层数较多的模型中，选择激励函数时就要小心了，因为可能会出现梯度消失和梯度爆炸的问题

  • 一般来说，卷积神经网络推荐relu，循环神经网络推荐Relu或者tanh

  • 常见的几种激励函数F

sigmoid ：

  • 求导$\frac{d}{dx}\sigma(x)=\sigma(1-\sigma)$

  • 优点：

    • 压缩至[0,1]，保证数据稳定波动幅度小
  • 缺点：
    • 梯度消失或者爆炸
    • 输出不是0均值——导致，如果输入为正，那么导数总为正，反向传播总往正方向更新，如果输入为负，那么导数总为负，反向传播总往负方向更新，收敛速度缓慢
    • 运算量较大

tanh 函数

  • 优点：

    • 解决了Sigmoid函数非0均值的问题，将函数压缩至区间[-1, 1]之间

  • 缺点：

    • 函数在两端的饱和区梯度趋近于0，当反向传播时容易出现梯度消失或梯度爆炸
    • 运算量大

Relu

  • 优点：

    • 梯度不饱和，收敛速度快
    • 减轻反向传播时梯度弥散（消失）的问题
    • 由于不需要进行指数运算，因此运算速度快、复杂度低

  • 缺点：

    • 输出不是0均值(zero-centered)
    • 对参数初始化和学习率非常敏感，设置不当容易造成神经元坏死现象，也就是有些神经元永远不会被激活(由于负部梯度永远为0造成)
